## 💡 重要发现：内置工具能力

### Realtime API 原生支持

官方文档显示 `step-audio-2` 系列模型**内置**以下能力：

1. **web_search 工具** ✅
   - 无需自己实现搜索接口
   - 直接调用即可

2. **retrieval 工具** ✅
   - 内置知识库检索
   - 支持向量化存储

3. **工具调用 (Tool Call)** ✅
   - 原生支持
   - 可以调用自定义函数

**影响评估**:
```
原计划: 需要自己实现检索系统
实际情况: step-audio-2 已内置 retrieval 工具

简化方案:
- 阶段 1 的"知识图谱"功能可以直接使用内置 retrieval
- 阶段 2 的"网络搜索"可以直接使用内置 web_search
- 大幅降低开发复杂度！
```
