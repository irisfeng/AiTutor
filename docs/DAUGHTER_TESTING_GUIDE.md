## 💡 使用建议

### 日常学习

1. **课后复习**：问老师讲过的知识点
2. **作业不会**：直接问具体问题
3. **预习新课**：问下节课要学的概念
4. **写作文**：用文学学科找素材和思路

### 学科选择

- **历史课复习**：选择📜历史
- **数学作业**：选择📐数学
- **古诗词鉴赏**：选择📖文学

1. **直接问问题**，不要说"你好"
2. **具体一点**，不要问太大范围
3. **问不懂的**，重复问直到理解
4. **多学科**，可以随时切换学科
